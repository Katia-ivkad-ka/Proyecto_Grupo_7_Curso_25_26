# Proyecto_Grupo_7_Curso_25_26
## Proyecto grupal - DAM Grupo 7 - Proyecto Intermodular

### ¿Qué es este proyecto?
Se propone una aplicación para la gestión de la facturación 
en el sector de la climatización y aire acondicionado.

### Objetivo
Permitir a los técnicos registrar la información del trabajo 
directamente desde el lugar donde se realiza, generando facturas 
en tiempo real sin necesidad de papeleo.

### Contenido del repositorio
- `memoria.pdf` — Memoria del proyecto 
- `presentacion.txt` — Enlace a la presentación
- `diagramas/` — Diagramas de análisis y diseño
- `ui/` — Diseño de pantalla principal
